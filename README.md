# FreMo: Frequency-Domain Multi-Modality Transportation Modeling
**This implementation showcases our FreMo model.** 
## Installation Dependencies
``` 
Python 3 (>= 3.6; Anaconda Distribution)

PyTorch (>= 1.6.0) 

Numpy >= 1.17.4

Pandas >= 1.0.3
```

python traintest_FreMo.py cuda_id
